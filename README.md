## 📚 内容概览

### A1 计算机基础
- CPU 架构与工作原理（控制单元、ALU、寄存器、总线）
- Fetch-Decode-Execute 循环详解
- 内存层次结构（Cache、RAM、ROM、虚拟内存）
- 操作系统基础（进程管理、内存管理、文件系统）

### A2 数据结构与算法
- 基础数据结构（数组、链表、栈、队列）
- 树结构（二叉树、二叉搜索树、AVL 树）
- 常见排序算法（冒泡、选择、插入、归并、快速排序）
- 搜索算法（线性、二分搜索）

### A3 数据库系统
- 关系型数据库设计原理
- SQL 语法（SELECT、JOIN、聚合函数）
- 数据库规范化（1NF、2NF、3NF）
- ER 图建模

### A4 网络与互联网
- OSI 七层模型
- TCP/IP 协议栈
- HTTP/HTTPS 协议
- 网络安全基础

### B 编程实践
- 面向对象编程（OOP）
- 数据验证与错误处理
- 算法复杂度分析（Big O）
- 软件开发方法论

## 🚀 使用指南

### 在线阅读（推荐）

访问 **[在线文档站点](https://michaelmjhhhh.github.io/I-HATE-CMS/)** 获得最佳阅读体验：

- 🎨 精美的 Material Design 界面
- 🔍 强大的全文搜索功能
- 📱 完美的移动端适配
- 🌓 支持亮色/暗色主题切换
- 📊 Mermaid 流程图实时渲染

### 本地查看

```bash
# 克隆仓库
git clone https://github.com/michaelmjhhhh/I-HATE-CMS.git
cd I-HATE-CMS

# 方式 1: 直接查看 Markdown 文件
# 使用 Typora、VSCode 或任意 Markdown 编辑器打开
open "I HATE CMS_FINAL.md"

# 方式 2: 本地运行文档站点
python3 -m venv venv
source venv/bin/activate
pip install mkdocs-material mkdocs-mermaid2-plugin pymdown-extensions
mkdocs serve
# 访问 http://127.0.0.1:8000
```
